# Unimask Playground

A simple playground to demonstrate the functionality of the [Unimask](https://github.com/dschewchenko/unimask) library.

## Features

This playground demonstrates:

- Vue directive usage (`v-unimask`)
- Vue composable usage (`useUnimask`)
- React hook usage (`useUnimask`)
- Different mask types (phone numbers, postal codes)

## Running the Playground

1. Install dependencies:

```bash
cd playground
bun install
```

2. Start the development server:

```bash
bun run dev
```

3. Open your browser and navigate to http://localhost:5173

## Examples Included

1. **Vue Directive Example**: Phone number mask using the `v-unimask` directive
2. **Vue Composable Example**: UK postal code mask using the `useUnimask` composable
3. **React Hook Example**: Phone number mask using the `useUnimask` React hook

Feel free to modify the examples or add your own to test different mask configurations!
